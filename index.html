Movie Recommender Application

This app gives content-based movie recommendations using tags. It uses the process of Text Vectorization using the 'Bag of Words' technique and applies Cosine Similarity to find the closest vectors and generate accurate recommendations w.r.t genre, director, overview, cast and crew. It displays the top 5 movie recommendations using the model and their posters using an API request to tmdb. The project uses the python libraries numpy, pandas, streamlit, nltk, requests and tmdbv3api.
